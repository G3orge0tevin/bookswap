# Bookswap

## Project Info

This project is a web application built using modern web technologies to facilitate book swapping. It leverages a component-based architecture with reactive state management.

## Technologies Used

This project is built with:

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**

## Getting Started

To work on this project locally, you need to have **Node.js & npm** installed.

### Local Development

Follow these steps to run the project on your machine:

1.  **Clone the repository:**
    ```bash
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd bookswap
    ```

3.  **Install the necessary dependencies:**
    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```
    This will start the server with auto-reloading and an instant preview.
